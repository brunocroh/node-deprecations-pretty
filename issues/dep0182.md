## Description

This codemod should add explicit authTagLength option when using short GCM authentication tags. It's useful to migrate code that uses GCM ciphers with short authentication tags without explicit length specification.

It should add explicit authTagLength option to GCM cipher operations with short authentication tags. It should handle both CommonJS and ESM imports. It should preserve existing cipher options while adding required authTagLength. It should ensure security by making tag lengths explicit.

## Examples

Case 1

Before:

```js
const crypto = require("node:crypto");
const cipher = crypto.createCipher("aes-256-gcm", "secret");
cipher.setAuthTag(Buffer.alloc(12)); // 12-byte tag without explicit length
```

After:

```js
const crypto = require("node:crypto");
const cipher = crypto.createCipher("aes-256-gcm", "secret", { authTagLength: 12 });
cipher.setAuthTag(Buffer.alloc(12));
```

Case 2

Before:

```js
const crypto = require("node:crypto");
const cipher = crypto.createCipherGCM("aes-256-gcm", key, iv);
const tag = cipher.getAuthTag(); // May be short without explicit length
```

After:

```js
const crypto = require("node:crypto");
const cipher = crypto.createCipherGCM("aes-256-gcm", key, iv, { authTagLength: 12 });
const tag = cipher.getAuthTag();
```

Case 3

Before:

```js
const crypto = require("node:crypto");
function createSecureCipher(algorithm, key) {
  return crypto.createCipher(algorithm, key);
}
```

After:

```js
const crypto = require("node:crypto");
function createSecureCipher(algorithm, key, options = {}) {
  if (algorithm.includes("gcm") && !options.authTagLength) {
    options.authTagLength = 16; // Default safe length
  }
  return crypto.createCipher(algorithm, key, options);
}
```

Case 4

Before:

```js
import crypto from "node:crypto";
const decipher = crypto.createDecipher("aes-128-gcm", password);
decipher.setAuthTag(shortTag);
```

After:

```js
import crypto from "node:crypto";
const decipher = crypto.createDecipher("aes-128-gcm", password, { authTagLength: shortTag.length });
decipher.setAuthTag(shortTag);
```
